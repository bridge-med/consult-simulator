# ConsultSim

診察オペレーション設計シミュレーター

1診/2診体制、曜日ごとの診察時間・休憩設定、枠構成（診察・注射・回診・月1・MRI結果・新患BF）のカスタマイズができる、中小医療機関向けの運用設計ツール。

🌐 **Live**: https://bridge-med.github.io/consult-simulator/

## できること

- **体制切替**: 1診 / 2診 をワンクリック比較
- **曜日編集**: 月〜日、それぞれの診察時間・午前/午後枠数・昼休憩を自由設定
- **グループ編成**: 「月火金を同じ設定」など、ユーザー定義のグループで同期編集
- **30分枠の中身設計**: 診察・注射・回診などを組み合わせ、バッファ時間を可視化
- **需要vs供給サマリー**: 月間の患者需要に対して、設計した枠がどれだけカバーできるかを即確認
- **回診シミュレーション**: 物療患者数と回診枠数から、1回あたり/1名あたりの所要時間を試算

## 使い方

ブラウザで https://bridge-med.github.io/consult-simulator/ を開くだけ。データはブラウザローカルで処理され、サーバーに送信されません。

## 開発

単一HTMLファイル（`index.html`）。フレームワークなし、外部依存はGoogle Fontsのみ。

```bash
# ローカルで開く（ブラウザで直接開くだけでOK）
open index.html
```
